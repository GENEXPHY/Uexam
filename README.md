# Uexam - Platform Ujian Online

Website ujian online dengan tema biru gelap, fitur keamanan, dashboard guru, dan sistem leaderboard.

## Fitur Utama

### Untuk Siswa
- Login dengan username dan password
- Mengerjakan soal pilihan ganda
- Leaderboard waktu pengerjaan
- Penjelasan soal yang dijawab salah
- Proteksi anti tab baru
- Responsive design

### Untuk Guru
- Login khusus
- Upload soal pilihan ganda dengan media
- Upload penjelasan soal
- Hapus progress ujian siswa
- Lihat riwayat ujian

## Teknologi
- Next.js 14 + React 18
- Tailwind CSS
- Firebase (Firestore, Auth, Storage)
- Deployed di Vercel
